# 🧠 Assistente de Resumos com IA

## 🎯 Objetivo

Este projeto utiliza IA para transformar textos em:

- Resumos
- Tópicos principais
- Perguntas de revisão

## 🚀 Tecnologias

- Python
- OpenAI API

## 📌 Como usar

1. Instale dependências:
   pip install -r requirements.txt

2. Configure sua API no .env

3. Execute:
   python app.py

## 💡 Exemplo de uso

Entrada:
Texto longo sobre inteligência artificial...

Saída:

- Resumo
- Tópicos
- Perguntas

## 📊 Aprendizados

- Engenharia de prompts melhora muito os resultados
- Estruturar saída facilita o uso prático
- IA funciona melhor com instruções claras
